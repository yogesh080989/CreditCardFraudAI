# CreditCardFraudAI

Enterprise-grade AI-powered Credit Card Fraud Detection System with Explainable AI and Agent Evaluation Framework.

---

# Project Overview

CreditCardFraudAI is an end-to-end machine learning framework developed as the Capstone Project for the Executive Diploma in Machine Learning & AI.

The objective is to detect fraudulent credit card transactions using multiple machine learning algorithms while following enterprise software engineering practices.

Unlike traditional notebook-based implementations, this project is designed as a modular Python application with reusable components for data validation, preprocessing, model training, evaluation, explainability, and AI-assisted decision support.

---

# Business Objective

Credit card fraud causes billions of dollars in financial losses every year.

The objective of this project is to:

- Detect fraudulent transactions accurately
- Reduce false positives
- Explain every prediction using Explainable AI
- Build an enterprise-ready reusable ML framework
- Provide AI-assisted fraud investigation
